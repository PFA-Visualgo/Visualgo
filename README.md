# Visualgo
A PFA project from 2023-2024 students of ENSEIRB-MATMECA.

# How to use

Option 1: 
- Type `make`

Option 2 (with automatic updates of the website):
- Use VsCode extension "Live Server", start and join http://localhost:5500/src/
